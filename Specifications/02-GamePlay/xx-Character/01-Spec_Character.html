Specification : Character Engine

The aim of this document is to detail the specification for the Character Engine

Characters

Overview

  • Specification

Specification

A Character is a game entity with which the player may either be or have interactions.
This game engine is supposed to offer the player to be a character which may interact with Non Player Characters, have some attributes and skills, fight, and so on.
A Character is a main part of the gameplay.

ICharacter

An ICharacter represents a character for fight engine.
It has multiple attributes that are changed from external composents (ISkill, IEffect...).

BasicCharacter

This class is the basic representation of a character for the application.
